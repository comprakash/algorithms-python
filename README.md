## Big O Notation

| Big-O    | Name         |
|----------|--------------|
| 1        | Constant     |

| n        | Linear       |
| nlog(n)  | Log Linear   |
| n^2      | Quadratic    |
| n^3      | Cubic        |
| 2^n      | Exponential  |
